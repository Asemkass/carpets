Premium Carpets – Carpet Cleaning Website

A modern responsive landing page for a professional carpet cleaning service.
The website presents company services, cleaning process, gallery, and contact information in a clean and minimal design.

The project is built using HTML and CSS with responsive layout and multilingual support.

Features

Responsive design (desktop and mobile)

Modern UI with clean layout

Multilingual interface

Russian

Kazakh

English

Hero section with call-to-action

About company section

Cleaning process visualization

Gallery of results

Contact section with WhatsApp button

Modern footer with social links

Technologies Used

HTML5

CSS3

Google Fonts (Montserrat, Open Sans)

Font Awesome icons
